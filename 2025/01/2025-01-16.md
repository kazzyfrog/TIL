## React dev tool

また React Developer Tools では、state の状態が変化したときに、再レンダリングされるコンポーネントを可視化することができます。

「View settings」を選択し、「Highlight updates when components render.」にチェックを入れます。

https://zenn.dev/yskn_sid25/articles/415c7e7a81ac6a

再レンダリング時ハイライトの設定が便利！
