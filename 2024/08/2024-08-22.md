## 今日の学び

### デフォルトを、ダイナミックインポートする場合

> 返されたオブジェクトから "default" のキーを再構築し、名前を変更する必要があります。

```js
(async () => {
  if (somethingIsTrue) {
    const {
      default: myDefault,
      foo,
      bar,
    } = await import("/modules/my-module.js");
  }
})();
```

知らなかった！
